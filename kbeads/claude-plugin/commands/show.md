---
description: Show detailed information about an issue
argument-hint: <issue-id>
---

Show detailed information about a kbeads issue.

If argument is provided, use it as the issue ID. Otherwise ask for the issue ID.

Run: `kd show <id>`

Present the full issue details including:
- Title, type, status, priority
- Description and notes
- Dependencies (blocks/blocked-by)
- Labels
- Assignee and owner
